FinalBasement Portable for Mac
==============================

To run:
  Double-click START_FINALBASEMENT.command

Or open Terminal in this folder and run:
  ./FinalBasement

Leave the Terminal window open while using the site.
Close the Terminal window to stop the site.

The app opens at:
  http://localhost:8501

If macOS blocks it:
  Right-click START_FINALBASEMENT.command
  Click Open
  Click Open again

If it still says permission denied, open Terminal in this folder and run:
  chmod +x START_FINALBASEMENT.command FinalBasement

If macOS says the app is damaged/quarantined, run:
  xattr -dr com.apple.quarantine .
