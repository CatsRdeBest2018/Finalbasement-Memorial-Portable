FinalBasement Portable
======================

To run:
  Double-click START_FINALBASEMENT.bat

Or double-click:
  FinalBasement.exe

Leave the black window open while using the site.
Close the black window to stop the site.

The app opens at:
  http://localhost:8501

This folder contains its own Python runtime inside the executable bundle.
The other computer does not need Python installed.

If Windows SmartScreen appears, click More info, then Run anyway.
